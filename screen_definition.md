# Kikillionaire 화면 정의서 (Screen Definition)

아이폰 12 mini 규격(375 x 812 pt)에 최적화된 Kikillionaire 가계부 웹앱의 화면 구성안입니다. 모든 화면의 최하단에는 빠르게 이동할 수 있는 **하단 네비게이션 바(Bottom Navigation Bar)**가 상시 노출됩니다.

---

## 📱 1. 전체 화면 구성 요약 (Tab Navigation)

| 탭 순서 | 화면명 | 경로 (Route) | 주요 기능 / 구성 요소 |
| :--- | :--- | :--- | :--- |
| **1** | **Quick Entry** | `/` (Home) | **오직 수동 입력 폼(FastEntryForm)만 단독 노출** (앱 켜자마자 빠른 입력용) |
| **2** | **Assets** | `/dashboard` | 가용/비가용 자산 목록, EUR/KRW 환율 변환 토글 스위치 |
| **3** | **Insights** | `/insights` | 예산 진행 바, 런웨이(생존일) 계산, 일일 권장 지출액 가이드 |
| **4** | **History** | `/history` | 전체 거래 내역 조회 (수입/지출/이체 리스트) |

---

## 🔍 2. 각 화면별 상세 정의

### 화면 1: Quick Entry (경로: `/`)
- **목적**: 앱 접속 시 클릭 최소화하여 즉시 지출을 기록하는 홈 스크린.
- **구성 요소**:
  - 지출(Expense), 수입(Income), 이체(Transfer) 세부 탭.
  - 카테고리(지출 시), 출금 자산, 입금 자산(이체 시), 금액 입력 필드.
  - "Add Transaction" 등록 버튼.
  - *다른 요소(대시보드, 리스트 등)는 일절 배제하여 심플하게 유지.*

### 화면 2: Assets (경로: `/dashboard`)
- **목적**: 내 전체 자산 규모와 통화별 잔액을 한눈에 확인하고 변환하는 화면.
- **구성 요소**:
  - 실시간 EUR/KRW 기준 환율 정보 표시.
  - **EUR/KRW 글로벌 토글 스위치** (스위치 변경 시 하단 금액 모두 즉시 실시간 환산).
  - **가용 자산 목록**: 한국 예금, N26, 현금(EUR), 트래블월렛, SC은행 잔액.
  - **비가용 자산 목록**: 한국 적금(일할 계산된 실시간 누적 이자 포함), 한국 주식(야후 파이낸스 실시간 시세 반영 평가금 및 수익률 포함).
  - **자산 정보 수동 수정 (Modal Popup)**:
    - 자산 목록에서 특정 자산 항목을 클릭하면 모달 팝업 오픈.
    - 일반 자산: 현재 잔액(Balance) 직접 수정 가능.
    - 주식 자산: 총 매수 원금(Balance) 및 보유 주식 수량(Quantity) 수정 가능.
    - 적금 자산: 총 납입 원금(Balance) 및 이율(Extra) 수정 가능.
    - 수정 후 저장 시 구글 시트의 해당 자산 행(Row)을 실시간으로 업데이트.

### 화면 3: Insights (경로: `/insights`)
- **목적**: 남은 가용 자산과 소비 속도를 기반으로 예산을 관리하고 생존 일수를 계산해 경고하는 화면.
- **구성 요소**:
  - **Available Assets** (가용 자산 총액) 요약.
  - **Runway 안내**: "Runway: X Months" 또는 "Runway: Y Days" 형태로 생존 예상 일수 표기.
  - **런웨이 경고 바**: 런웨이 남은 기간에 따른 소진 속도 인디케이터(Progress Bar) 표시 및 긴축 지출 권장 메시지.
  - **Monthly Spend**: 최근 30일 누적 지출액 표시.
  - **Daily Recommended Spend**: 남은 자산 기반 오늘 하루 권장 지출 상한액 안내.

### 화면 4: History (경로: `/history`)
- **목적**: 과거 거래 이력을 스크롤하며 검토하는 거래 내역서 화면.
- **구성 요소**:
  - 유형별(Expense/Income/Transfer) 텍스트 컬러 구분 (지출은 적색, 수입은 녹색, 이체는 기본 텍스트 색상).
  - **필터 토글 버튼** 탑재 (`All`, `Expense`, `Income`, `Transfer` 필터링 기능 지원).
  - 스크롤을 통해 과거 입력한 수동 데이터를 확인할 수 있는 오버플로우 스크롤 영역.

## 🛠️ 네비게이션 설계 (Bottom Navigation Bar)
- **높이**: 약 `60px` 고정.
- **아이콘 및 라벨**:
  - ⚡ `Quick` (홈 `/`)
  - 🏦 `Assets` (대시보드 `/dashboard`)
  - 📊 `Insights` (통계 `/insights`)
  - 📜 `History` (내역 `/history`)
- **디바이스 고정 크기(375x812)** 내에서 하단에 고정(`position: absolute; bottom: 0; width: 100%`)되며, 스크롤 영역은 이 네비게이션 바 바로 위까지만 흐르도록 제한하여 아이폰의 앱 네비게이션 사용성과 동일하게 맞춥니다.
